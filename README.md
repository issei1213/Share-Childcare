# アプリ名
## Share Childcare

## 概要実装機能
  育児専用のマッチングアプリケーション

## 制作背景
  スクールのカリキュラム以外で学習した内容を、本アプリ作成時に使用し開発を実施。また、育児専用のマッチングアプリとし、今まで作成してきなかったカテゴリのアプリを開発を試みた。

## 実装機能
- ユーザー管理機能
- 依頼機能
- 通知機能
- チャット機能
- コメント・評価機能(実装予定 )
